AI Career Copilot Agent

Overview

AI-powered career assistant built using n8n, Gemini AI, Google Sheets, and JavaScript.

The workflow helps job seekers analyze resumes against job descriptions and generate personalized recommendations.

Features

* ATS Resume Scoring
* Matching Skills Identification
* Missing Skills Analysis
* Resume Improvement Suggestions
* ATS Optimized Resume Generation
* Interview Question Generation
* Automated Results Tracking

Tech Stack

* n8n
* Gemini AI API
* Google Sheets
* JavaScript
* REST APIs
* Prompt Engineering

Workflow

1. Read Resume Data
2. Read Job Description
3. Generate ATS Analysis
4. Generate Resume Optimization
5. Generate Interview Questions
6. Store Results in Google Sheets

Sample Output

* ATS Score
* Matching Skills
* Missing Skills
* Resume Suggestions
* Optimized Resume
* Interview Questions and Answers
